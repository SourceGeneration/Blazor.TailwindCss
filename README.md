# Blazor.TailwindCss

## Installing

```powershell
Install-Package SourceGeneration.Blazor.TailwindCss -Version 1.0.0-beta4.250201.1
```

```powershell
dotnet add package SourceGeneration.Blazor.TailwindCss --version 1.0.0-beta4.250201.1
```
